## 1.1.二分查找

```
public static int binarySearch(int[] a, int key)
{
    int lo = 0, hi = a.length - 1;
    while(lo <= hi)
    {
        int mid = lo + (hi -lo) / 2;
        if (key < a[mid]) hi = mid - 1;
        else if (key > a[mid]) lo = mid + 1;
        else return mid;
    }
    return -1;
}
```

## 1.2.Union-find算法

### 1.1.1.Quick-find算法

#### 数据结构（N）

数组长度为N的$id[]$;

如果p和q有相同的id，则把p和q连接。

#### 查找（1）

检查p和q是否有相同的id。

#### 合并（N）

合并p和q只需把id[p] = id[q]

* 改变的数值过多，数值多的时候计算量大

### 1.1.2.Quick-union算法

#### 数据结构（N）

数组长度为N的$id[]$;

id[i]是i的父节点

#### 查找(N)

检查p和q是否有相同根节点root

#### 合并(N)

合并p和q即把p的id的根指向q的id的根

### 1.1.3.Weighted quick-union算法

#### 数据结构(N)

数组长度为N的$id[]$;

id[i]是i的父节点

sz[i]计算对象到根的距离

#### 查找(lg N)

检查p和q是否有相同根节点root

#### 合并(lg N)

把根小的树连接到根大的树上

更新sz[]数组

#### 路径压缩

将树展平每个根节点都直接指向root
