# 3. 查找

## 3.1. 符号表

符号表就是把一个键和一个值联系起来，存储键值的数据结构。支持两种操作：插入(put)，即将一组新的键值对存入表中,查找(get)，即根据给定的键得到相应的值。

```java
public class ST<Key, Value>
{
    ST() //创建一个符号表
    void put (Key key, Value val) //将键值对存入表中
    Value get(Key key) //获取键key对应的值
    
    void delete(Key key) //删除键值
    {
        put(key, null);
    }
    
    boolean contains(Key key) //键key中是否有值
    {
        return get(key) != null;
    }

    boolean isEmpty() //表是否为空
    {
        return size() == 0;
    }
    int size() //表中键值对数量
    Iterable<Key> keys() //表中所有键的集合
}
```

### 3.1.1. 无序链表中的顺序查找

```java
// 算法3.1 顺序查找
public class SequentialSearchST<Key, Value>
{
    private Node first;

    private class Node
    { //链表节点定义
        Key key;
        Value val;
        Node next;

        public Node(Key key, Value val, Node next)
        {
            this.key = key;
            this.val = val;
            this.next = next;
        }
    }

    public Value get(Key, key)
    {//查找给定键，返回相对应值
        for (Node x = first; x != null; x = x.next)
            if (key.equals(x.key))
                return x.val;
        return null;
    }

    public void put(Key key, Value val)
    {//查找给定键，找到则更新，否则建立行结点
        for (Node x = first; x != null; x = x.next)
            if (key.equals(x.key))
            {
                x.val = val;
                return;
            }
        first = new Node(key, val, first);
    }
}
```

### 3.1.2. 有序数组中的二分查找

数据结构是一对平行的数组，一个存储键一个存储键。

```java
//算法3.2 二分查找
public class BinarySearchST<Key extends Comparable<Key>, Value>{

    private Key[] keys;
    private Value[] vals;
    private int N;
    
    public BinarySearchST(int capacity){
        keys = (Key[]) new Comparable[capacity];
        vals = (Value[]) new Object[capacity];
    }

    public int size(){
        return N;
    }

    public Value get(Key key){
        if(isEmpty()) return null;
        
        int i = rank(key);
        
        if(i<N && keys[i].compareTo(key) == 0) return vals[i];
        else return null;
    }

    public int rank(Key key){//有序数组的二分查找,用于查找键
        int lo = 0; hi = N-1;
        
        while (lo <= hi){
            int mid = lo + (hi - lo) / 2;
            int cmp key.compareTo(keys[mid]);
            if (cmp < 0) hi = mid - 1;
            else if(cmp > 0) lo = mid + 1;
            else return mid;
        } 
        return lo;
    }

    public void put(Key key, Value val){
        int i = rank(key);
        // 找到相应键，替换值
        if(i < N && keys[i].compareTo(key) == 0){
            vals[i] = val; return
        }
        // 没有相应值，新建键值
        for (int j = N; j > i; j--){
            keys[j] = keys[j-1];
            vals[j] = vals[j-1];
            keys[i] = key;
            vals[i] = val;
            N++;
        }
        
    }
}
```

```java
private Node deleteMin(Node x){
    if (x.left == null) return x.right;
    x.left = deleteMin(x.left); //递归找到最深的左值
    x.N = size(x.left) + size(x.right) + 1; //更新总节点数
    return x;

public void delete(Key key){
    root = delete(root, key);
}

private Node delete(Node x, Key key){
    if (x == null) return null;
    int cmp = key.compareTo(x.key);

    if (cmp < 0) x.left = delete(x.left, key);
    else if (cmp > 0) x.right = delete(x.right, key);
    else{
        if (x.left == null) return x.left;
        if (x.right == null) return x.right;
        //t：要删除的点
        Node t = x;
        //x：替代点
        x = min(t.right); //见3.2.2， 返回最小值，为后继节点
        //替代点的右值是最接近t的右值
        x.right = deleteMin(t.right);
        x.left = t.left;
    }
    x.N = size(x.left) + size(x.right) + 1;
    return x;
}
```

## 3.3. 平衡查找树

### 3.3.1. 查找树

为保证树的平衡性在2-节点（一个键两条链接）基础上引入3-结点（两个键和三条链接）

### 3.3.2. 红黑树

3-节点的实现：树中链接分为两种类型，红链接把两个2-节点连接起来构成一个3-节点；黑链接则是树中的普通连接。

#### 3.3.2.1. 特点

1. 红链接均为左连接
2. 没有任何一个节点和两个红链接相连
3. 该树是完美黑色平衡，即任意空链接到根节点的路径上黑链接数量相同

#### 3.3.2.2. 颜色表示

将链接的颜色保存在节点的Node数据类型的布尔变量color中，如果链接是红色就是true，是黑色就是false。

```java
private static final boolean RED = true;
private static final boolean BLACK = false;

private class Node{
    Key key;
    Value val;
    Node left, right;

    boolean color;

    Node(Key key, Value val, int N, boolean color){
        this.key = key;

private boolean isRed(Node x){
    if(x = null) return false;//默认是黑
```

#### 3.3.2.3. 旋转

##### 左旋

一条红色的右链需要被转换成左链接

```java

Node rotateLeft(Node h){//h是需要被左旋的点，x是原本的右点
```
