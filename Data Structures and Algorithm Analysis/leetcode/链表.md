## 237. 删除链表中的节点

由于不删除结尾节点，所以只需要把删除节点的值和指向用下一个值替代即可

```java
/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) { val = x; }
 * }
 */
class Solution {
    public void deleteNode(ListNode node) {
        node.val = node.next.val;
        node.next = node.next.next;
    }
}
```

## 876. 链表的中间结点

使用快慢指针，快指针一次next两次，慢指针一次，当快指针到末尾时慢指针正好一半

```java
class Solution {
    public ListNode middleNode(ListNode head) {
        ListNode slow = head;
        ListNode fast = head;
        while (slow!=null&&fast!=null&&fast.next!=null){
            slow = slow.next;
            fast = fast.next.next;
        }
        return slow;
    }
}
```

## 206. 反转链表

### 循环

```java
class Solution {
    public ListNode reverseList(ListNode head) {
        ListNode header = null;
        ListNode node = head;
        ListNode prev = null;
        while(node != null){
            ListNode nodeNext = node.next;
            if(nodeNext == null) header = node;
            node.next = prev;
            prev = node;
            node = nodeNext;
        }
        return header;
    }
}
```

### 递归

```java
public ListNode reverseList3(ListNode pHead){
	if(pHead==null || pHead.next == null){ //如果没有结点或者只有一个结点直接返回pHead
		return pHead;
	}
	ListNode pNext = pHead.next; //保存当前结点的下一结点
	pHead.next = null; //打断当前结点的指针域
	ListNode reverseHead = reverseList3(pNext); //递归结束时reverseHead一定是新链表的头结点
	pNext.next = pHead; //修改指针域
	return reverseHead;
}
```
