# 智力题

## 292. Nim游戏

巴什博奕，n%(m+1)!=0时，先手总是会赢

```java
class Solution {
    public boolean canWinNim(int n) {
        return n%4!=0;
    }
}
```

# 原理题

## 371. 两整数之和

 a ^ b是无进位的相加； a&b得到每一位的进位

```java
class Solution {
    public int getSum(int a, int b) {
        return b == 0 ? a : getSum(a^b,(a&b)<<1); 
    }
}
```
